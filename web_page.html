目的

本プロジェクトでは今まで独自のホームページを所有しておらず，外部への情報発信手段が極めて少なかった．
「外部への効率的な情報発信手段の獲得」と「実践的なWebプログラミングを通しての技術向上」を目的とし，今回ホームページの作成を行った．

開発環境

型番 EPSON Endeavor NA601E(カスタマイズ済)
OS Windows 10 Pro 64bit
CPU Intel Core i5-3337U (1.8GHz)
メモリ PC3-12800 SODIMM 16GB (8GB×2)
主記憶装置 Crucial MX500 500GB
ソフトウェア Microsoft Visual Studio Code

開発にはEndeavor NA601Eにパーツ換装等で独自のカスタマイズを加えた物を使用した． Visual Studio Codeのバージョンは開発期間中に何度か更新されているため，ここでの記載は省略する．
